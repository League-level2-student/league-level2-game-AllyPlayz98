Zodiac Scramble

by: Olivia Hicks

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decided to make this game because the zodiac is something that is very interesting to me. Also, I don't think a lot of people know the order of the zodiac signs, so I thought it could be a good learning tool as well.

2. What I Learned

I learned that convenience is more important than I originally thought. Before I made my final changes, the player had to restart the entire program to play the game again. Also, I made it so the scrambled letters at the top of the screen change their order each time the game is played.

3. Architecture of My Game

There are five parts to this game: the menu, the instruction, the game, the fail, and the end screens. The menu, instruction, fail, and end screens should be pretty self explanatory. The game screens consist of the scrambled letters of each zodiac sign and a text box to type in the unscrambled sign.

4. Technical Implementation

I am very proud of the fact that the scrambled letters randomize their order each time the game is played. It took me a very long time for me to do this, so I'm very pleased with this accomplishment. I had to change all the code I had initially written to get the images of the letters on the screen for it to work, and that took a while for all twelve signs. Even though it was a minor change, I am proud of all the work that I put into it.

5. Most Challenging Part

I think the most challenging part of this project was getting the text box on the game screens. I tried two different methods to get it to appear, and they ended up overlapping. I tried to remove all of the code for the first one, but later in the coding process I found more that interfered with the text box I had kept. It took a little bit of time to make sure I only had one text box, but I figured it out.

6. Future Fixes and Updates

Overall, I'm very happy with the way the game turned out. The only think I would change would probably be the background. I think having it be one color is a little boring, so I might have made a custom background to make it more interesting.
